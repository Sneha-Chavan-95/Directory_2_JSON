# Directory_2_JSON

🗃️Directory to JSON

This CLI application is designed to convert directory structure to a JSON file.

Sneha Chavan

🥷 Contributor

💻 Create a Virtual Environment

# Install the virtualenv module
pip install virtualenv

# Create the virtual Environment named venv
python -m venv venv
or
virtualenv venv

# Activate the Virtual Environment
venv\Scripts\activate
🚀 Install the Application

# Install the Directory 2 JSON app
pip install -e.
😎 Install the Extras

pip install -e ".[dev,test]"
✔ Running Tests To run the tests, refer the following

❄ Flake8

flake8 src test
🧪 Pytest

pytest -v
💘 mypy

mypy src test
🧹 pylint

pylint src test
🦅 vulture

vulture src test
Features
Convert the directory structure to JSON file.
